## Como converter DOCX para PDF

1. Abra o arquivo DOCX no Microsoft Word ou LibreOffice
2. Vá em "Arquivo" > "Salvar Como" ou "Exportar como PDF"
3. Salve o arquivo PDF nesta pasta com o nome correspondente

Ou use ferramentas online como:
- https://www.ilovepdf.com/docx_to_pdf
- https://convertio.co/docx-pdf/
